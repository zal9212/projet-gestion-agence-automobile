# Teranga Auto ERP

Teranga Auto ERP est une solution de gestion de location de voitures complète, moderne et sécurisée, conçue spécifiquement pour le marché sénégalais. Elle allie une interface client haut de gamme à un tableau de bord administratif puissant.

---

## Fonctionnalités Clés

### Pour les Clients
*   **Recherche Intuitive** : Moteur de recherche multicritère (Marques, Modèles, Dates).
*   **Expérience Mobile** : Design "App-like" optimisé pour un usage fluide sur smartphone.
*   **Réservation en Ligne** : Processus de réservation rapide avec calcul automatique des tarifs.
*   **Signature Électronique** : Validation des contrats directement sur l'écran.
*   **Espace Personnel** : Gestion du profil, des favoris et historique des réservations.

### Pour l'Agence (Admin & Agents)
*   **Dashboard Décisionnel** : Statistiques en temps réel (CA, flotte, réservations).
*   **Planning Gantt** : Vue graphique interactive de l'occupation des véhicules sur 30 jours.
*   **Gestion de Flotte** : CRUD complet avec upload multiple et sélecteur de logos de marques.
*   **Gestion des Rôles** : Accès différenciés pour les Administrateurs et les Employés (Agents).
*   **Contrats PDF/Print** : Génération automatique de contrats professionnels avec QR Code.
*   **Alertes Maintenance** : Notifications pour les assurances arrivant à expiration.

## Installation

### Prérequis
*   Un serveur local (XAMPP, WAMP, MAMP) ou un hébergement web.
*   PHP 8.1 ou supérieur.
*   MySQL / MariaDB.

### Étapes
1.  **Cloner le projet** dans votre répertoire `htdocs` ou `www`.
2.  **Importer la base de données** : Utilisez le fichier `voiture_v2.sql` dans PHPMyAdmin.
3.  **Configuration** : Modifiez les accès à la base de données dans le fichier `app/config.php`.
4.  **Lancer l'application** : Accédez à l'URL via votre navigateur (ex: `http://localhost/pjgv`).
